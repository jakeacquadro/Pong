=== PONG ===

I remade the classic arcade game with 2 modes: Player vs Player and Player vs Computer.

To run: 
	Use command:
	"python3 App.py"


== CONTROLS ==

1 PLAYER MODE:
	PLAYER:
		UP: UP ARROW KEY (↑)
		DOWN: DOWN ARROW KEY (↓)


2 PLAYER MODE:
	PLAYER 1:
		UP: A
		DOWN: Z

	PLAYER 2:
		UP: UP ARROW KEY (↑)
		DOWN: DOWN ARROW KEY (↓)
